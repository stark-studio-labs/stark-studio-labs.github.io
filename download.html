One-click setup for new modders

Download the app, open it, and click "Auto Setup." That's it. Stark Labs detects your game, creates a safe mod folder structure, and installs sensible defaults.

1

Download the app

Grab the installer above for your operating system. Open the .dmg (Mac) or run the .exe (Windows).

2

Click "Auto Setup"

The app finds your Sims 4 installation, creates a backup point, and configures everything automatically.

3

Browse & install

Pick any mod from the built-in catalog. One click to install. One click to undo. No file management ever.

4

Launch & play

Start The Sims 4 normally. Stark Labs runs quietly in the background, watching for conflicts.

Pick exactly what you want

Use the mod picker to select individual gameplay layers. Each runs independently — install Economy without Politics, or Smart Sims without Drama. Your call.

1

Open Mod Picker

Launch Stark Labs and go to the "Layers" tab to see all available gameplay mods.

2

Select your layers

Toggle on Economy, Smart Sims, QoL — whatever combination you want. The conflict checker runs live.

3

Configure each mod

Click the gear icon on any mod to adjust settings — difficulty sliders, feature toggles, behavior tuning.

4

Apply & play

Hit "Apply Changes." Stark Labs handles file placement, load order, and cleanup. Launch your game.

Full control via CLI and GitHub

Clone repos, install via pip, use the hot-reload dev server. Everything is MIT-licensed and built for extensibility.

1

Install the DevKit

One pip command gets you the full CLI with project scaffolding, build tools, and the dev server.

2

Scaffold a mod

Run stark init my-mod to get a typed project with hot-reload, tests, and packaging ready to go.

3

Develop with hot-reload

Run stark dev — changes deploy into The Sims 4 instantly. No restart, no export, no waiting.

# Install the Stark DevKit
$ pip install stark-devkit

# Create a new mod project
$ stark init my-mod --template gameplay
✓ Created my-mod/ with gameplay template

# Start developing with hot-reload
$ cd my-mod && stark dev
⚡ Hot-reload server running on port 8400
⚡ Connected to The Sims 4 — drop in and test
