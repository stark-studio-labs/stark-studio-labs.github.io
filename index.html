See It In Action

Modding should feel modern.

Clean APIs. Type safety. Hot reload. The Stark Framework gives you the developer experience Sims modding has always needed.

my_first_mod.py 
from stark_framework import mod, on_event, inject
from stark_framework.sims import Sim, Interaction

@mod("my_awesome_mod", version="1.0.0")
class MyMod:

    @on_event("sim.interaction.start")
    def on_interaction(self, sim: Sim, interaction: Interaction):
        if interaction.is_social:
            sim.add_buff("confident", duration=240)
            self.log(f"{sim.name} feels extra confident!")

    @inject("services.sim_info_manager")
    def boost_all_sims(self, manager):
        for sim in manager.get_all():
            sim.skills.painting.add_xp(500)

Decorators handle the boilerplate. Type hints give you autocomplete. Event hooks let you react to anything in the game. And @inject gives you direct access to EA's internal services — no more hunting through decompiled code.
