How to Install Sims 4 Mods:
Complete Step-by-Step Guide

Installing mods for the first time is easier than it looks. This guide walks you through every step — from flipping the right switch in your game settings to watching your first mod load up. No experience needed.

📄 Beginner-friendly — PC & Mac

The Sims 4 has one of the best modding communities in gaming — but if you've never installed a mod before, it can feel confusing. Where do files go? What's a .package file? What if I break my game?

Don't worry. This guide covers everything. By the end, you'll have mods running and a system for managing them going forward.

Before you start

The Sims 4 is free to play, so mods work for everyone — paid packs not required. Mods only affect your save files, so you can always remove them if something goes wrong.

Step 1: Enable Mods in Your Game Settings

Before any mod will show up in your game, you need to turn mods on. EA has them disabled by default.

  1. 1

    Open The Sims 4 and go to the main menu

    You don't need to load a save. Just get to the screen where you'd normally pick your household.

  2. 2

    Click the three dots (⋯) → Game Options → Other

    On PC it's the three dots in the top-right corner. On Mac, same location.

  3. 3

    Check both boxes: "Enable Custom Content and Mods" and "Script Mods Allowed"

    The first checkbox enables .package files (most mods and custom content). The second one enables .ts4script files, which are more powerful gameplay mods. You want both checked.

  4. 4

    Click "Apply Changes" and restart the game

    The game needs a full restart for the settings to take effect. Close it completely and reopen.

Step 2: Find Your Mods Folder

Your Mods folder is where you'll put every mod you download. It's created automatically when you first run the game.

On PC (Windows)

Open File Explorer and paste this path into the address bar:

Documents\Electronic Arts\The Sims 4\Mods

Or navigate there manually: This PC → Documents → Electronic Arts → The Sims 4 → Mods

On Mac

Open Finder, click Go in the top menu, then Go to Folder, and paste:

~/Documents/Electronic Arts/The Sims 4/Mods

Tip

Bookmark or pin this folder — you'll come back to it every time you install a new mod.

Step 3: Download a Mod

Most mods are hosted on sites like CurseForge, ModTheSims, a creator's Patreon, or directly from GitHub. When you click download, you'll get either a .zip file or a direct .package / .ts4script file.

Safety note

Only download mods from trusted sources. Avoid random file-sharing sites. Reputable mods come from Patreon, CurseForge, ModTheSims, or the creator's official Discord.

Step 4: Extract the Files (if needed)

If your download is a .zip file, you need to unzip it first. Right-click the file and choose Extract All (Windows) or double-click it (Mac). You should end up with one or more files ending in .package or .ts4script.

Important: Don't put the ZIP file into your Mods folder — extract it first. The game can't read ZIP files.

Step 5: Place the Files in Your Mods Folder

Drag the extracted .package and/or .ts4script files directly into your Mods folder. You can organize them into subfolders — the game will find them as long as they're no more than 5 folders deep.

Organization tip

Create subfolders like Mods/Gameplay/, Mods/UI/, and Mods/Stark/ to keep things tidy. It makes troubleshooting much easier later.

Step 6: Launch the Game and Verify

Start The Sims 4. When it loads, you should see a notification saying something like "X mods were found." Click that notification or go to Game Options → Other to see a list of all your installed mods.

If your mod appears in that list with a green checkmark, it's active and working.

Frequently Asked Questions

What's the difference between a .package file and a .ts4script file?

.package files contain data — new objects, textures, trait values, custom content. .ts4script files contain Python code that changes how the game actually behaves. Most gameplay-heavy mods use .ts4script. Both need the Script Mods setting enabled.

My mod isn't showing up. What's wrong?

Three most common causes: (1) You forgot to restart the game after enabling mods. (2) The files are still inside a ZIP. (3) The mod is in a subfolder more than 5 levels deep. Check all three first.

Will mods break my game?

Well-maintained mods from trusted creators are very safe. The main risk is game updates — EA patches can break mods temporarily until the creator updates them. Get in the habit of checking for mod updates after every Sims 4 update.

Can I use mods on Mac?

Yes, everything above works on Mac. The only difference is the folder path (shown in Step 2).

How many mods can I have?

Technically hundreds, but the more mods you have, the slower your load times. Most players with large mod collections use 50–200 mods comfortably.
